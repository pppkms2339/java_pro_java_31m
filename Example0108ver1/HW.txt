Описать класс "цветная точка". Для точки задаются координаты и цвет. Цвет описывается с помощью трех составляющих (красный, зеленый, синий). Предусмотреть различные методы инициализации объекта с проверкой допустимости значений. Допустимым диапазоном для каждой составляющей является [0, 255]. Описать свойства для получения состояния объекта и метод изменения цвета.
Написать программу, демонстрирующую все разработанные элементы класса.
Разрабатываемый класс должен, как правило, содержать следующие элементы: скрытые поля, конструкторы с параметрами и без параметров, методы, свойства.
